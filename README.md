# 文学博客 - 岁月缱绻 葳蕤生香

一个温馨优雅的个人文学博客，展示读书感悟、小说连载和生活分享。

## ✨ 特色功能

- 📚 **好书推荐** - 精选文学作品推荐
- 📖 **读书感悟** - 深度阅读思考和感悟
- 📝 **小说连载** - 原创小说作品展示
- 🌸 **生活分享** - 温柔的生活记录
- 🔍 **智能搜索** - 支持关键词搜索和分类筛选
- 📱 **响应式设计** - 完美适配桌面端和移动端

## 🚀 快速部署

### 方法一：使用部署脚本（推荐）

**Linux/Mac 用户：**
```bash
./deploy.sh
```

**Windows 用户：**
```cmd
deploy.bat
```

### 方法二：手动部署

1. **安装依赖**
```bash
pnpm install
```

2. **编译项目**
```bash
pnpm run build
```

3. **部署 dist 文件夹**
将生成的 `dist` 文件夹内容上传到任意静态托管服务。

## 🌐 推荐托管平台

- **Vercel** (推荐) - https://vercel.com
- **Netlify** - https://netlify.com  
- **腾讯云静态网站托管** - 国内访问快
- **阿里云 OSS** - 稳定可靠
- **GitHub Pages** - 免费使用

## 🛠️ 技术栈

- **前端框架**: React 18 + TypeScript
- **构建工具**: Vite
- **样式方案**: Tailwind CSS
- **路由管理**: React Router
- **UI组件**: Radix UI
- **部署方式**: 静态网站托管

## 🎨 设计理念

采用编辑杂志式的文学风格设计：
- **色彩**: 奶茶色 (#C8A882) + 淡紫色 (#B8A1C8)
- **字体**: 思源宋体，营造书香气息
- **布局**: 简洁优雅，注重阅读体验
- **氛围**: 温馨静谧，温柔有力量

## 📖 使用说明

1. **内容管理**: 目前使用静态数据，如需添加/修改内容，请编辑 `public/data/` 目录下的 JSON 文件
2. **图片更换**: 将新图片放入 `public/imgs/` 目录，并更新对应的数据文件
3. **样式调整**: 修改 `src/index.css` 和组件样式文件

## 🔧 本地开发

```bash
# 安装依赖

# 启动开发服务器
pnpm dev

# 预览构建结果
pnpm preview
```
